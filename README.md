# OptiCube One-Page Game Showcase

Simple single-page website for presenting OptiCube games, including a featured title, portfolio cards, filters, and quick studio stats.

## Run locally

Open `index.html` in any browser.

If you prefer a local server:

```bash
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).

## Project files

- `index.html` - page structure and content blocks
- `styles.css` - visual design, layout, animations, and responsive behavior
- `script.js` - game data, filters, and stats rendering
